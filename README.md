# react移动H5练习项目

> **前言:**
>
> 此项目是类似于头条新闻的练习项目
>
> 技术栈：react + redux + react-redux + react-router + typescript + axios + antd-mobile
>
> 账号：13911111111密码：246810
>
> 后端接口可能会失效。

> **安装环境：**
>
> 需要在node环境下，在package.json所在的文件夹中npm i下载依赖，下载完成后npm run start启动项目在`localhost:3000`即刻访问
